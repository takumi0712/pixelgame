# Requirement

pyxel 1.4.3

# Installation


```bash
pip install pyxel
```

# Usage
```bash
git clone https://github.com/takumi0712/pixelgame.git
cd game
python main.py
```

# 操作方法
<ul>
<li>左に移動 = A</li>
<li>右に移動 = D</li>
<li>ダッシュ = AorD + shift</li>
<li>スライディング = ダッシュ中に control</li>
<li>しゃがみ = S</li>
<li>左クリック = 攻撃</li>
<li>ジャンプ = space</li>
<li>ジャンプ攻撃 = ジャンプ中に左クリック</li>
</ul>
